# Manage Tasks app

a simple todo app or task managing app which implemented with firebase and reactjs

## Run app

To run the app install it first with

```bash
  npm install
```

## start the app

This project is used by the following companies:

```bash
  npm start
```

## techs

- ReactJs

- Context Api for theme

- Sass

- Bootstrap
